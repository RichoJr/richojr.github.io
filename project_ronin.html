Ronin

Ronin is a co-op multiplayer escape room experience where two players need to work together to solve puzzles and escape. Each puzzle has its own room and on its completion the players are given a puzzle piece which they can use on a pedestal in the central room of the dungeon to unlock the final door.


This was a student project where myself and another designer took on a combined designer and programmer role. The project involved us creating an initial concept based on a client brief where we had to develop an escape room experience where each puzzle and room could be remade in a real life scenario.

Platform:

Windows PC

Engine:

Unity

Language:

C#

Team Size:

6

Duration:

3 months

Roles & Contributions

Concepting & Documentation

  • Creating a game concept and puzzles based off a brief which needed the project to be recreated in the real world
  • Working on a GDD with the fellow designer, making regular updates on parts we were working on


Level Designer

  • Planning out dungeon layout using external level tool(Dungeon Scrawl)
  • Greyboxing level environments from initial designs
  • Tweaking room layouts based on gameplay issues and re-designs
  • Setting up QTE sequence using a QTE handler made by other designer


Technical Designer

  • Scripting mechanism functionality for related puzzles(pulley, crank)
  • Creating puzzle functionality for four puzzles (memory-based puzzle, shadow puzzle, coordinate finding puzzle, rotary puzzle)
  • Creating firefly particle for puzzle pieces


UI/UX Designer

  • Creating menu functionality and layout
