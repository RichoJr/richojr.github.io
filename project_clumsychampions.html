Clumsy Champions

Clumsy Champions is a wacky, physics-based party game where players engage in medieval combat with the goal of being the first to win three rounds. The catch of this game is, at the end of each round all of the players who lost are given a choice of permanent upgrades to tilt the balance in their favour.

Platform:

Windows PC (Steam)

Engine:

Unreal Engine 5 (UE5)

Language:

UE5's Blueprint

Team Size:

7

Duration:

5 months

Roles & Contributions

Documentation Manager

  • Working on and keeping a GDD, Production Plan, and Project Tracking Tool up-to-date throughout project


Technical Designer

  • Creating the settings menu layout and complete functionality
  • Creating early framework of the parry system while programmer was occupied
  • Creating an occlusion mask for when player's line of sight would be obstructed
  • Updating the potion selection system to use a timer based system
  • Updating the environmental hazards system sometimes target players and increase their spawn chance based on the number of players


Sound Designer

  • Locating SFX from a sound library that fit with the theming of the project
  • Creating and allocating sound cues and audio mixers
  • Creating moments when the audio cue needs to play


UI/UX Designer

  • Creating the UI framework across the project
  • Creating and connecting functionality towards player HUDs & menu sequences


QA

  • Creating test plans to check each section of the project is running correctly
  • Using test data from play testers and test plans to make appropriate changes towards gameplay feel and functionality
