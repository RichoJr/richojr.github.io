Davo's Trasheria

Davo's Trasheria was made for the Ubisoft Australia 2024 Game Jam. It's a fast-paced trash recyclement sim, where the player takes trash that's been dropped onto a conveyor belt and turns it into a profitable item. The trash must be removed from the conveyor belt before it reaches the end and falls into a furnace. If they fail to do so it'll produce pollutants that lower the player's daily score.

Platform:

Windows PC (itch.io)

Engine:

Unity

Language:

C#

Team Size:

6

Duration:

2 weeks

Roles & Contributions

Concepting

  • Researching how different types of trash are recycled in Australia and incorporating these techniques into gameplay
  • Creating a brief narrative at the start of the game which explains how and why they are in the factory


Level Designer

  • Grey boxing the factory layout with a focus on the best distance for each piece of machinery and length of the conveyor belt before it reaches the furnace


Technical Designer

  • Creating gameplay timer and conveyor belt functionality using C# scripting


Gameplay Designer

  • Assigning all trash objects to their machinery and workbench items
  • Tweaking level(day) escalating difficulty as they progress(conveyor belt speed, type of trash to drop, time which trash type takes to be recycled)
  • Tweaking item sale price to allow of the purchase of appropriate upgrades for each day


Lighting Artist

  • Creating a night time effect using Unity's directional light
  • Setting up factory spot lights to help create a visual distance from each piece of machinery
